# Reference

## `/config/meta.json`

```json
{
  "$schema": "https://developers.booqable.com/schemas/app-0-2-0.json#",
  "id": "liquid_conf",
  "version": "1.0.0",
  "name": "MyApp",
  "author": "Booqable",
  "app_store": { /* AppStoreInfo */ },
  "plans": [ /* ... */ ],
  "configuration_page": { /* ConfigurationPage */ },
  "tracking_script": { /* TrackingScript */ },
  "theme_blocks": [ /* ... */ ],
  "oauth": { /* OauthConfig */ },
  "base_url": "https://example.com",
  "payments": { /* PaymentsConfig */ }
}
```

| Name | Type | Required | Description |
|------|------|----------|-------------|
| `$schema` | string | true | Schema version for the app configuration. |
| `id` | string | true | Unique identifier for the app, used internally. |
| `version` | string | true | Version number or status of the app. |
| `name` | string | true | Internal name for the app, not shown to end users. |
| `author` | string | true | Name of the app developer or company. |
| `app_store` | [`AppStoreInfo`](#reference-appstoreinfo) | true |  |
| `plans` | array of [`AppPlan`](#appplan) | false | Subscription plans for the app. |
| `configuration_page` | [`ConfigurationPage`](#reference-configurationpage) | false | Optional configuration page content (banners, forms, guides, iframes). Omit the key when the app has no configuration page. |
| `tracking_script` | [`TrackingScript`](#reference-trackingscript) | false | Settings for apps that provide tracking scripts meant to be installed in store pages. |
| `theme_blocks` | array of [`ThemeBlock`](#themeblock) | false | Website blocks provided by this app for use in the website editor. |
| `oauth` | [`OauthConfig`](#reference-oauthconfig) | false | OAuth integration for apps that require Booqable to grant access on behalf of the user. |
| `base_url` | string | false | Base URL to use when providing relative URLs elsewhere in this configuration file. |
| `payments` | [`PaymentsConfig`](#reference-paymentsconfig) | false | Payment integration for apps that process charges, authorizations, and refunds. |

## `FormBlockField`

```json
{
  "name": "lorem ipsum",
  "type": "text",
  "label": { /* LocalizableString */ },

  "description": { /* LocalizableString */ },
  "placeholder": { /* LocalizableString */ },

  "multi": true,
  "pattern": ".*\.mydomain\.com/.*"
}
```

| Name | Type | Required | Description |
|------|------|----------|-------------|
| `name` | string | true |  |
| `type` | one of: `text`, `textarea`, `number`, `password`, `quantity`, `select`, `price`, `email`, `color`, `time`, `datepicker`, `checkboxGroup`, `buttonGroup`, `radio`, `radioGroup`, `checkbox`, `timeSelect`, `phone`, `contentEditor`, `url`, `font`, `code`, `search`, `duration`, `multiInput`, `dateButton`, `timeButton`, `timeslots`, `durations`, `range` | true | Input type; must match a type supported by the back-office form (InputField). |

| `required` | boolean | false |  |
| `description` | [`LocalizableString`](#reference-localizablestring) | false |  |
| `placeholder` | [`LocalizableString`](#reference-localizablestring) | false |  |
| `options` | array of [`SelectFieldOption`](#selectfieldoption) | false | Required for select type. |
| `multi` | boolean | false | Allow multiple selection for select fields. |
| `pattern` | string | false | Regex pattern for validating text/textarea fields. The value must match this pattern. |

## `ThemeBlock`

```json
{
  "name": { /* LocalizableString */ },
  "template": { /* AssetPath */ },
  "allowed_sections": [ /* ... */ ],
  "cookies_required": true,
  "settings_iframe_url": { /* LocatableUrl */ },
  "settings": { /* ... */ }
}
```

| Name | Type | Required | Description |
|------|------|----------|-------------|
| `name` | [`LocalizableString`](#reference-localizablestring) | true | Display name for the block in the website editor. |
| `template` | [`AssetPath`](#reference-assetpath) | true | Path to the Liquid template file for the block. |
| `allowed_sections` | array of string | true | Section types where this block may be placed. |
| `cookies_required` | boolean | false | Display a notice in this block's settings about the block not rendering without website visitors providing cookie consent. |
| `settings_iframe_url` | [`LocatableUrl`](#reference-locatableurl) | false | Embed an iframe in the website editor sidebar for this block's settings, instead of the usual controls. |
| `settings` | object | false | Which settings are available for this block in the website editor. |
## `ThemeBlockSetting`

```json
{
  "type": "header",
  "content": { /* LocalizableString */ },
  "label": { /* LocalizableString */ },
  "placeholder": { /* LocalizableString */ },
  "default": "lorem ipsum",
  "min": 42,
  "max": 42,
  "step": 42
}
```

| Name | Type | Required | Description |
|------|------|----------|-------------|
| `type` | one of: `header`, `divider`, `text`, `textarea`, `number`, `color`, `checkbox` | true | Type of the setting field. |
| `content` | [`LocalizableString`](#reference-localizablestring) | false | Text for header type. |
| `label` | [`LocalizableString`](#reference-localizablestring) | false | Display label for the setting. |
| `placeholder` | [`LocalizableString`](#reference-localizablestring) | false | Hint text inside the input. Applies to text, textarea. |
| `default` | `string` \| `number` | false | Default value. |
| `min` | number | false | A minimum value for number typed fields. |
| `max` | number | false | A maximum value for number typed fields. |
| `step` | number | false | A step increment for number typed fields. |
